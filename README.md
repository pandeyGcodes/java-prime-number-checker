<h1>java-prime-number-checker</h1>
This project is a Java program that checks whether a number is prime or not. It uses an efficient method and handles invalid inputs. It helped me understand basic programming concepts, logic building, and writing clean and simple code. Project Report: Java Prime Checker

Introduction

This project focuses on building a simple yet practical Java application that determines whether a given number is prime or not. Prime numbers are fundamental in mathematics and play an important role in areas like cryptography, algorithms, and data structures.
Problem Statement The problem addressed in this project is how to efficiently check whether a number is prime while handling invalid inputs properly and maintaining clean and structured code.
Objective The objectives include implementing an efficient algorithm, handling invalid inputs, following best coding practices, and ensuring readability.
Approach and Methodology The program checks divisibility only up to the square root of the number, improving efficiency. It validates input, handles edge cases, and outputs the result.
Features Implemented Prime detection, input validation, optimized algorithm, clean code structure, and exception handling.
Key Decisions Used square root optimization, added exception handling, kept program simple, and separated logic clearly.
Challenges Faced Handling invalid inputs, balancing optimization and simplicity, and ensuring readability.
Results The program correctly identifies prime numbers, handles invalid inputs, and runs efficiently.
Learning Outcomes Learned practical application of concepts, importance of optimization, safe input handling, and writing maintainable code.
Conclusion This project shows that even simple problems can benefit from structured approaches and optimization
